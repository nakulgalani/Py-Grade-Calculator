###### JARVIS Grade Calculator

A futuristic Iron Man-inspired Grade Calculator built using Python and Streamlit.

## Features

* Modern JARVIS-themed UI
* Grade calculation based on marks
* Input validation
* Displays:

  * Grade A (90+)
  * Grade B (80–89)
  * Grade C (70–79)
  * Grade D (60–69)
  * Grade E (40–59)
  * Fail (Below 40)
  * Invalid Input (Outside 0–100)

## Technologies Used

* Python
* Streamlit
* HTML/CSS Styling

## Run Locally

Install Streamlit:

```bash
pip install streamlit
```

Start the application:

```bash
streamlit run app.py
```
